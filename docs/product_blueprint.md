# 产品蓝图：Court Booking System

## 1. 愿景

打造一个高效的网球场可用性检查工具，旨在为 Angus Glen Tennis Centre 的网球爱好者提供实时的场地信息。系统将自动查询并通知用户最新的可用时间段，帮助他们做出预订决策。

## 2. 目标用户

- **网球爱好者**: 经常在 Angus Glen Tennis Centre 打球，希望快速锁定理想时间段的个人或团体。
- **家庭与朋友**: 希望安排团体活动，需要协调多人时间并提前预订场地的用户。
- **教练与学员**: 需要定期预订场地进行训练的网球教练。

## 3. 核心功能

| 功能模块 | 描述 | 状态 |
| --- | --- | --- |
| **场地可用性检查** | 自动查询指定日期所有网球场的可用时间段。 | ✅ 已实现 |
| **实时 SMS 通知** | 通过 Twilio 服务，将最新的场地可用性信息实时推送到用户的手机。 | ✅ 已实现 |
| **会话与认证管理** | 自动处理与 PerfectMind 预订系统的会话认证，确保 API 请求的有效性。 | ✅ 已实现 |
| **Web 用户界面** | 提供一个直观的 Web 界面，方便用户查看场地可用性状态。 | 🚧 待开发 |

## 4. 技术栈

- **后端语言**: Python
- **依赖管理**: `uv`
- **API 交互**: `requests`
- **通知服务**: Twilio SMS API
- **数据存储**: `.json` (场地信息), `.env` (配置)
